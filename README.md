# EV Adoption Forecasting

## Overview

This project analyzes historical Electric Vehicle (EV) registration data from Washington State to forecast future EV adoption trends. The objective is to support infrastructure planning by predicting future electric vehicle demand using historical growth patterns and regional data.

## Dataset

The dataset includes monthly vehicle registration records from January 2017 to February 2024, categorized by county, vehicle type, and electric vehicle adoption metrics.

Key features:
- Date
- County
- Vehicle Primary Use
- Battery Electric Vehicles (BEVs)
- Plug-In Hybrid Electric Vehicles (PHEVs)
- Electric Vehicle Total
- Non-Electric Vehicle Total
- Total Vehicles
- Percent Electric Vehicles

## Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Tasks

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- EV adoption trend visualization
- Feature engineering
- Regression-based forecasting
- Future EV demand prediction

## Outcome

Developed a forecasting model to estimate future EV adoption based on historical registration trends, supporting data-driven planning for charging infrastructure and sustainable transportation initiatives.
